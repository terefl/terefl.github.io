Judopedia

Se trata de una aplicación móvil enfocada en el judo, que se organiza en tres apartados principales: fundamentos, técnicas y caídas. En el apartado de fundamentos se incluye un glosario especializado con términos propios del judo, acompañados de sus significados y definiciones detalladas. En la sección de técnicas, el contenido se encuentra dividido en tres categorías: judo de pie, judo de suelo y katas, abarcando cada una de ellas las técnicas correspondientes de manera estructurada. Finalmente, en el apartado de caídas se explican las formas adecuadas de ejecutarlas, siguiendo los principios técnicos del judo para garantizar tanto la seguridad como la efectividad. Esta aplicación está diseñada como una herramienta integral para facilitar el aprendizaje y la práctica de este arte marcial.

Lenguajes: Java.

Cosmos

Se trata de una página web dedicada al sistema solar, que presenta información resumida y clara sobre este tema. Además, cuenta con la funcionalidad de traducir los textos al inglés, lo que la hace accesible a un público más amplio. Esta página fue desarrollada como parte de un curso en colaboración con dos compañeros, combinando esfuerzos para lograr un diseño funcional y educativo.

Lenguajes: HTML, CSS y Javascript

Felinos

Se trata de una de las primeras páginas web que diseñé, la cual incluye información sobre varias razas de felinos. En ella se presenta una tabla que muestra el estado de conservación de cada especie, acompañada de noticias interesantes relacionadas con dichas razas. Además, en el menú principal se encuentra un formulario que permite a los usuarios enviar información sobre la raza que deseen.

Lenguajes: HTML y CSS
